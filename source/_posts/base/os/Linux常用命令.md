# 写在前面

当你不会使用一个命令时，可以用man查看命令帮助



# 文件操作相关

## cd

全称是change directory ，意思是改变目录

## ls



单词list的缩写，展示目录下面的文件和文件夹信息

## mkdir

全称是make directories，就是创建目录的意思

## touch

用于创建新文件或更新已有文件的访问和修改时间戳。

## rm

单词remove的缩写，删除文件系统文件、目录、设备文件、符号链接等

# 运维

## lsof

查看端口被哪个进程占用了

lsof -i :4000

## kill

强制杀掉4250进程

kill -9 4250
